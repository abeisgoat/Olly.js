Olly.js Examples

    To use Olly.js without feeding it an element
    you can replace the element argument with any boolean value:

<script>
var embedResults = olly.embed("http://vimeo.com/78276911", true);
</script>

Output

<script>
var embedResults =
    {
        scripts: Array[0],

    }
</script>
